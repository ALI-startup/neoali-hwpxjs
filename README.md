# neoali-hwpxjs

A fork of @ssabrojs/hwpxjs ([link](https://github.com/ssabro/hwpxjs)) with improved spacing handling for Pandoc-generated HWPX. 

## Features
- Correct spacing between inline elements
- Compatible with Pandoc HWPX output
